## 1.2 Фильтр Гаусса
Задача фильтра Гаусса - сглаживать изображение, уменьшая шум и устраняя ненужные детали, которые могут мешать анализу. Это низкочастотный фильтр, который сглаживает неравномерные значения пикселей изображения, обрезая самые высокие значения. Каждый пиксель заменяется взвешенным средним его окружения внутри окна размера K х K, приложенного в рассматриваемом пикселе. Чем ближе соседние пиксели к текущему, тем больший "вклад” они дают в ответ.

# 2. Морфологические операции
## 2.1 Эрозия
Эрозия или сужение. Данная операция представляет собой логическую операцию коньюнкции. То есть если на исходном изображении в окрестности точки, в том числе и она сама, все пиксели равны 1, то данная точка на выходном изображении будет равна 1, если нет то 0. Данная операция позволяет убрать лишние одиночные пиксели на изображении, полученные после бинаризации.

Суть бинаризации в следующем, представим, что у нас есть исходное изображение I, и мы хотим выделить, на нем все пиксели яркость которых больше
определенного порога. В результате все пиксели значение яркости которых будет больше установленного порога, будут выделены белым, а те которые меньше порога
черным.

## 2.2 Дилатация
Данная функция представляет собой логическую операцию дизьюнкции, то есть если на исходном изображении в окрестности точки, хотябы один пиксель равен 1, то данная точка на выходном изображении принимает значение 1, если нет, то 0. Данная функция позволяет востановить пексели внутри объектов, или части границ, которые были потерены в следствии бинаризации.

## 3.2 Выравнивание гистограмм
Выравнивание гистограммы предназначено для повышения контрастности изображения путем перераспределения яркостных значений таким образом, чтобы гистограмма интенсивностей была равномерно растянута по всему доступному диапазону (0-255).

Алгоритм:
- построение гистограммы частот 
- вычисление кумулятивной функции распределения (CDF)
- нормализация CDF 
- замена яркостей пикселей

Результат обработки после выполнения алгоритма гистограмма результирующего изображения становится равномерно распределенной по всему диапазону 0-255. Кумулятивная функция распределения (CDF) выровненного изображения представляет собой прямую линию, что соответствует идеальному равномерному распределению яркостей.

Гистограмма представляет из себя функцию h(x), которая возвращает суммарное количество пикселей, яркость которых равна x. Вернемся к функции распределения, упомянутой выше. Она определяется как сумма ячеек гистограммы от нуля до X:
